## 🛠️ Tecnologias de Gestão
Este sistema utiliza tecnologia de nuvem para garantir que não haja duplicidade de horários:
* **Interface:** Mobile-First (Otimizada para Celular)
* **Banco de Dados:** Firebase Firestore (Tempo Real)
* **Segurança:** Filtro por RG Militar e trava de vaga única.

---

## 📋 Instruções de Uso
1. Selecione a **Unidade de Destino** desejada.
2. O calendário liberará apenas as **datas oficiais** previstas em escala.
3. Escolha um dos **horários disponíveis** (slots de 15 minutos).
4. Após confirmar, o horário é removido automaticamente do sistema para evitar aglomerações.
